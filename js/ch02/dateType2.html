형변환 함수

데이터타입(자료형)을 변환하는 함수

1. String() : 데이터 타입을 문자타입로 변환하는 함수.
2. Number() : 데이터 타입을 숫자타입로 변환하는 함수.
3. Boolean() : 비논리형을 논리형으로 변환하는 함수.
- 논리형 : true, false 둘 중의 하나
- true : 문자열, 0을 제외한 모든 숫자(정수, 실수 포함)
- flase : 숫자 0 , null, undefined, NaN
4. toString() : 숫자를 문자로 변환하는 함수.
5. parseInt() : 문자를 정수로 변환하는 함수.
6. parseFloat() : 문자를 실수로 변환하는 함수.
