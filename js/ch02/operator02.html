연산자(operator)

3. 비교연산자 : >, >=, <, <=, ==(equal), !=(not equal), ===, !==
- ==, != : 값만 비교해서 true, false 결과를 출력
- ===. !== : 값과 데이터타입을 모두 비교해서 true, false 결과를 출력
- 우선순위 : (>, >=, <, <=) 연산자가 (==, !=) 보다 우선임.
- 우선순위 : (==,!=) 연산자가 (===,!==)보다 우선이다.
4. 논리연산자 : &&(논리 AND, 논리곱), ||(논리 OR, 논리합), !(논리 NOT, 논리 부정)
- 논리연산자의 결과는 논리값(true, false 둘 중의 하나)
- 비교연산자가 논리연산자보다 우선임. 비교 > 논리
- 우선순위 : 논리AND가 논리OR보다 우선임.
- 우선순위 : ! > && > ||
