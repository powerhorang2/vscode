연산자(operator)

1. 산술연산자 (+,-,*,/,%)
- / : 나누어서 몫, % : 나누어서 나머지
2. 증감연산자 (++, --)
- 증가연산자 : 변수의 앞 또는 뒤에 ++ 붙여 사용, 전위증가연산자, 후위증가연산자.
- 감소연산자 : 변수의 앞 또는 뒤에 -- 붙여 사용, 전위감소연산자, 후위감소연산자.
3. 비교(관계)연산자 : >, >=, <, <=, ==, !=, { ===, !== } (자바스크립트에서 추가)
- 결과가 논리값 형태로 표현, true 또는 false - >, >=, <, <=, ==, != : 값만 비교하여 true 또는 false의 값이된다. - === , !== : 자바스크립트에서 추가된 연산자, 값과 데이터타입을 비교하여 true 또는 false의 값이된다.
